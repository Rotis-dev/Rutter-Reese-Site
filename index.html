How I work

  1. Clarify audience + success criteria.
  2. Extract truth from SMEs (interviews, logs, tickets, shift notes).
  3. Draft fast, then tighten for clarity and scanability.
  4. Review cycle + versioning (changelog).
  5. Publish into a structure that’s easy to maintain.
