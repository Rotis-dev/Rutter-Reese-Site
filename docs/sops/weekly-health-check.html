SOP Routine Recordkeeping

Purpose

Perform a consistent weekly health check and capture a record of system status.

Audience

Ops/Support staff responsible for basic monitoring and recordkeeping.

Prerequisites

  • Access to monitoring dashboard(s) and system status page
  • Access to the weekly log (ticket, spreadsheet, or KB page)

Procedure

  1. Open the weekly log and create a new entry for this week.
  2. Verify telemetry freshness (last update within expected window).
  3. Review critical alarms for the last 7 days; list top 3 recurring alarms.
  4. Check communications uptime; record any outages.
  5. Check power/backup health indicators (if applicable); record status.
  6. Document any open follow-ups and assign an owner (name/role).
